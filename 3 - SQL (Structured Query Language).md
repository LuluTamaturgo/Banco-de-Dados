<h1 align="center"> SQL - Structured Query Language </h1>

 É a linguagem padrão para manipulação de dados em bancos de dados relacionais. Ela permite que os usuários interajam com os dados, criando, recuperando, atualizando e excluindo informações. A SQL é essencial para profissionais que trabalham com bancos de dados, como cientistas de dados, analistas de dados e desenvolvedores de software.

 
:heavy_check_mark:  Principais características da SQL: 

Características | Descrição
----------------|----------
Flexibilidade:  | A SQL pode ser utilizada em diversos Sistemas Gerenciadores de Banco de Dados (SGBDs) relacionais, como MySQL, PostgreSQL, Oracle e Microsoft SQL Server.
Facilidade de uso: | A sintaxe da SQL é relativamente simples e intuitiva, o que facilita o aprendizado e a utilização da linguagem.
Poderosa: | A SQL oferece uma ampla gama de comandos e funções para manipular dados de forma complexa e eficiente

:fast_forward:  Aplicações da SQL:

  * **Armazenar e organizar dados:** A SQL é utilizada para criar e gerenciar tabelas em bancos de dados relacionais.
  * **Consultar dados:** A SQL permite que os usuários recuperem informações específicas das tabelas do banco de dados.
  * **Analisar dados:** A SQL pode ser utilizada para realizar análises complexas dos dados armazenados no banco de dados.
  * **Atualizar e excluir dados:** A SQL permite que os usuários modifiquem ou removam informações do banco de dados.

    

:computer:  Tipos de comandos SQL - A SQL pode ser dividida em três categorias principais:

  * **Linguagem de Definição de Dados (DDL):**
      * Cria e modifica a estrutura do banco de dados, como tabelas, colunas e índices.
  * **Linguagem de Manipulação de Dados (DML):**
      * Insere, atualiza e exclui dados nas tabelas do banco de dados.
  * **Linguagem de Controle de Dados (DCL):**
      * Controla o acesso e as permissões dos usuários no banco de dados.
